## Häufige Fehler - Geo-Targeting

### ❌ Fehler 1: Falscher Länderfokus
**Falsch:** "Ich bin in Deutschland, aber optimiere auf .com"
**Richtig:** .de Domain + hreflang Tags + geo-IP targeting

### ❌ Fehler 2: Inkonsistente Lokale Informationen
**Falsch:** "Hamburg" auf Homepage, aber "Hambourg" im GMB
**Richtig:** Überall identische Name, Address, Phone (NAP)

### ❌ Fehler 3: Keine lokalen Backlinks
**Falsch:** Nur nationale/internationale Links
**Richtig:** Lokal.de, Branchenverbände, Hamburg-Blogs, Kammer-Verzeichnisse

### ❌ Fehler 4: Ignorieren von Review-Signalen
**Falsch:** GMB-Reviews ignorieren
**Richtig:** Aktiv Customers um Reviews bitten, auf jede Review antworten

### ❌ Fehler 5: Statisches lokales Content
**Falsch:** "Über uns" Page schreiben 1x und fertig
**Richtig:** Monatlich neuen lokalen Content (Events, News, Angebote)
